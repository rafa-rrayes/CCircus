# Circuit Circus

The Circuit Circus is the official SHDL circuit index. Here you can find a
collection of all the circuits created by the community and share your
circuits with others.

Install a library into an SHDL project with the `shdl` CLI (ships with
[PySHDL](https://pypi.org/project/pyshdl/)):

```bash
shdl add arith        # resolves against https://rafa-rrayes.github.io/CCircus/
```

Browse the catalog in `CATALOG.md`, or run the local site (`uv run ccircus`).
To publish a package of your own, see `BUILD_GUIDE.md`.
